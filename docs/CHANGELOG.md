# 更新日志

## [v2.0.0] - 2024-12-XX

### 🚀 性能优化版本

#### 新增功能
- 上下文增强：获取代码变更的完整上下文
- 行内评论：精确定位到具体代码行的评论
- 即时反馈：异步处理时立即返回处理状态

- 响应时间从分钟级降低到秒级（90%+提升）
- 支持并发处理多个审查任务

#### 用户体验改进
- 处理前显示详细步骤说明
- 评论触发时显示关键词确认
- 审查结果格式化输出

#### 技术改进
- 添加线程安全的缓存机制
- 优化错误处理和降级策略
- 改进代码结构和可维护性
